create_anonymization_table
Anonymization
Contents

  • Syntax
  • Usage
  • Example
About
A table is given with a column. Values are gathered from the columns and a 1-1 conversion table from them to anonymized entities is output.

Syntax
FUNCTION create_anonymization_table(in_tab VARCHAR, out_tab VARCHAR,
                                    id_col VARCHAR)
RETURNS VOID;
Parameters
in_tabName of input table.
out_tabName of output table.
id_colColumn to generate conversion table from.

Usage
The function creates new table (out_tab) with columns "id" and "anon_id", such that "anon_id" is the anonymized version of "id".

The anonymization process maps the original table values to random values. However, anyone who has the anonymization table can use the deanonymization process to map back to the original values.
