PDL Tools Revision History

PDL Tools 1.8 (packaged with SUgAR 1.0) – Current version

  • Add function for splitting data into train/test sets.
  • Add module for normalized cuts.
  • Add module for HITS algorithm.
  • Modified complete_linkage to support HAWQ.
  • Fixed auc function to handle division by zero errors.

PDL Tools 1.7 (packaged with SUgAR 1.0)

  • Add module kd_tree for KD-tree and its application to k-nearest neighbour search.
  • Add module pagerank.
  • Add grid search function for elastic net in grid_search module.
  • Add PDL Tools tutorials in Jupyter notebook.

PDL Tools 1.6 (packaged with SUgAR 1.0)

  • Add module one_vs_rest for multi-class classification.
  • Add module grid_search for grid searching functions.
  • Add module generic_utilities (includes multi-table summary and array utilities).

PDL Tools 1.5 (packaged with SUgAR 1.0)

  • Modifications in preparation to make PDLTools public. Removing module with pending IP (connected components).

PDL Tools 1.4 (packaged with SUgAR 1.0)

  • Added module for prediction metrics (regression+binary+multi-class).
  • Added module to balance datasets for classification.
  • Added SUgAR "format_glob" (globbing with user-defined transformation).
  • Added SUgAR support for user-extension tables.
  • Installer rewritten for database-safe upgrades.
  • More modules (e.g. SUgAR) now supported on HAWQ.

PDL Tools 1.3 (packaged with SUgAR 0.4.4)

  • Ported PDL Tools to work on HAWQ (1.2 and 1.3)
  • Modified pdlpack to accommodate installation on HAWQ
  • Added Complete Linkage algorithm.
  • Added multi-graph support for Connected Components algorithm.

PDL Tools 1.2.2 (packaged with SUgAR 0.4.4)

  • Added Stratified Sampling.
  • Added installation instructions for non-gpadmin install.

PDL Tools 1.2.1 (packaged with SUgAR 0.4.4)

  • Added Porter Stemming.
  • Path changes to pivotal.io domain.
  • Added 'version()' function.
  • Name change to PDL Tools.

DS Tools 1.2 (packaged with SUgAR 0.4.4)

  • Switched to Doxygen-based documentation (HTML+PDF)
  • Added Complex number support
  • MADlib schema can now be specified explicitly in install.

DS Tools 1.1.2 (packaged with SUgAR 0.4.3)

  • Added column globbing in SUgAR (pseudofunction 'glob').
  • Added ability to generate all-columns-but-one in pivoting.
  • Dos2unix on all files, to avoid '' in 'usage' printouts.

DS Tools 1.1.1 (packaged with SUgAR 0.4.2)

  • Fixed bugs in Connected Components to do with schema name scoping.
  • Installation now GRANTs USAGE on the schemas created.

  • DS Tools now packaged for both GPDB 4.2 and GPDB 4.3.
