session_split
Sessionization
Contents

  • Syntax
  • Usage
  • Example
About
A table is given with an id column and a time_stamp column. A "session" is a set of rows all associated with the same ID and all having consecutive time_stamps no more than time_out apart.

Syntax
FUNCTION session_split(in_tab VARCHAR, out_tab VARCHAR, id VARCHAR,
                       time_stamp VARCHAR, time_out INTERVAL)
RETURNS VOID;
Parameters:
in_tabName of input table.
out_tabName of output table.
idColumn to split by.
time_stampColumn of time stamps associated with row.
time_outTime interval for session split.

Usage
The function creates a new table, copying the original data and adding two new columns:
  • is_session_start - An indication of whether this is the first row in a new session.
  • session_no - An identifier for the session, such that (session_no,id) is a unique identifier.
