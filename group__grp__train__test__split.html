About
Given a table, an id column and split proportion, this function will create the desired train and test tables.

Syntax
FUNCTION train_test_split(sourceTable text, 
                          id text, 
                          trainSize double precision)
RETURNS VOID;
Parameters
sourceTableName of source table.
idName of the id column in the source table.
trainSizeDesired proportion of the train set.

Usage
The function creates a test table and a train table based on the trainSize parameter from the sourceTable. The output tables have sourceTable prefix followed by _test or _train.
